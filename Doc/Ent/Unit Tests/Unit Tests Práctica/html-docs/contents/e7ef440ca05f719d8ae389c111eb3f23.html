obtenerSegundoOperando

UMLOperation
Calculadora Project :: Model :: Parseador :: obtenerSegundoOperando

Description

Método para la obtención del segundo operador de la expresión mátemática.

Si la expresión de entrada es "4 + 7", este método debe devolver el entero 7

Parameters

Direction Name Type Description
in expresion String

Cadena de caracteres que representa la expresión matemática.
return segundoOperando Integer

Entero que representa el segundo operando de la operación extraído de la expresión matemática
