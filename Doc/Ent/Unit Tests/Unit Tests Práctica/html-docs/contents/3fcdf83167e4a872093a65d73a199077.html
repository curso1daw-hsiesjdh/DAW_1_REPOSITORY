Visibility Name Description
public sumar(sumando1: Integer, sumando2: Integer): Integer

Lleva a cabo la operación sumando1 + sumando2 y devuelve el resultado
public multiplicar(multiplicando1: Integer, multiplicando2): Integer

Lleva a cabo la operación multiplicando1 * multiplicando2 y devuelve el resultado
public restar(minuendo: Integer, sustraendo: Integer): Integer

Lleva a cabo la operación minuendo - sustraendo y devuelve el resultado
public dividir(dividendo: Integer, divisor: Integer): Double

Lleva a cabo la operación dividendo / divisor y devuelve el resultado. Si el operador divisor es igual a cero se debe lanzar la excepción BadOperatorException con el mensaje "División por 0. No se puede realizar la operación de división."
public computarExpresion(expresion: String): Double

Recibe por parámetro una cadena de caracteres del tipo "4 / -3" y en función del operador extraido de la expresión anterior se debe invocar al método sumar, restar, multiplicar o dividir con los parámetros extraidos de la expresion de entrada. Para este ejemplo, se debe invocar la función dividir con los parámetros 4 y -3, es decir: this.dividir(4, -3);
