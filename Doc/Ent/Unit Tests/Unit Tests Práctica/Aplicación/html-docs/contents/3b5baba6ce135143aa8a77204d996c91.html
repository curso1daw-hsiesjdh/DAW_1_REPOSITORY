Parseador

UMLClass
Calculadora Project :: Model :: Parseador

Description

Clase de utilidad empleada para descomponer una expresión matemática de la forma operando1 operador operando2 en cada una de sus partes.

Supondremos que la cadena de caracteres que representa a la expresión matemática está bien formada.
